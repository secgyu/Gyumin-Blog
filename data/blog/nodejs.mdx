## Node.js 설치하기 🚀

Node.js 설치 방법은 간단합니다.
먼저 Node.js 공식 홈페이지로 이동합니다. 아래의 주소로 이동 또는 구글에 nodejs라고 검색한 다음 최상단에 표시되는 사이트로 접근하면 됩니다.

"https://nodejs.org/ko"

설치 파일을 다운로드 했다면 실행 후 Node.js를 설치하면 됩니다.

## ✅ Node.js 설치 확인하기

Node.js 설치를 마쳤다면 명령어를 입력해 잘 설치 되었는지 확인해주세요.
터미널을 열어 다음 명령어를 입력합니다.

```
node -v
```

터미널에 node -v 명령을 입력하면 현재 자신의 PC에 설치된 Node.js 버전이 출력됩니다.  
만약 버전이 출력되지 않는다면 터미널을 새롭게 열어 동일하게 명령어를 입력합니다.  
또 다시 버전이 출력되지 않는다면 Node.js가 정상적으로 설치 되었는지 확인합니다.

## ✅ npm 설치 확인하기

```
npm -v
```

npm은 Node Package Manager로 Node.js 패키지를 설치, 삭제하고 버전을 업그레이드하는 기능을 제공합니다.
